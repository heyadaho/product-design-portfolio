## Overview

Our product suite had grown without a shared visual language. This project established a design system and component library used across three product teams.

## Problem

Inconsistent components and patterns led to higher dev effort and a fragmented user experience. Designers and engineers needed a single source of truth.

## Process

We audited existing UIs, defined tokens (color, type, spacing), built core components, and documented usage in Storybook. We ran adoption workshops with design and engineering.

## Solution

A documented design system with 30+ components, clear usage guidelines, and Figma libraries. Components are available in React and Vue.

## Outcomes

- Design-to-dev handoff time reduced
- Consistent patterns across all products
- New features now default to system components
