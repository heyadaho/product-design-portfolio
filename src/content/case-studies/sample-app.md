## Overview

This project explored how to simplify task management for busy teams. We started with user interviews and ended with a shipped mobile app and web dashboard.

## Problem

Teams were switching between multiple tools and losing context. We needed a single place to capture, assign, and track work.

## Process

We ran discovery workshops, created wireframes, and iterated with weekly usability tests. The design system was built in parallel with the first feature set.

## Solution

We shipped a mobile app with offline support and a responsive web dashboard. Key flows included quick-add, assignment, and status views.

## Outcomes

- Reduced time-to-log by 40% in internal testing
- Positive feedback on clarity of the assignment flow
- Design system adopted by two other product teams
